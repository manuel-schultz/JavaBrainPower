
# Difficulties
## Mathtrainer Level 1
Generates a math example that adds or subtracts 2 to 4 numbers between 0 and 100.
Examples:
```
87 + 29 + 24 - 43 = 97
92 + 49 = 141
42 + 16 - 62 = -4
18 - 39 - 28 + 34 = -15
```

## Mathtrainer Level 2
Generates a math example that adds or subtracts 3 to 8 numbers between -100 and 100.
Examples:
```
-79 - 88 + 0 - 85 - 86 = -338
35 + -92 - -40 - 50 + 76 - 88 = -79
1 + -45 + -43 + 83 = -4
9 + 82 - -94 + -33 - -63 = 215
```
